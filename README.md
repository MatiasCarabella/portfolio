## ğŸš¦ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/MatiasCarabella/portfolio.git

# Navigate to project directory

# Install dependencies
```
